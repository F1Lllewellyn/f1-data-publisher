# F1 No-Code Forecast Automation Buttons

Run this installer on Windows:

installer/RUN_F1_NO_CODE_FORECAST_AUTOMATION_BUTTONS_WINDOWS.bat

Then commit and push with GitHub Desktop.

After the push, GitHub Actions will show two simple workflows:

- F1 Forecast Automation - Safe Test Button
- F1 Forecast Automation - Run Now Button

Both require no fields to be filled in.
