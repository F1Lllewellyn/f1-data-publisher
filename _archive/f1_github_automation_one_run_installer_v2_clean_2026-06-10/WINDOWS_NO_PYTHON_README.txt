WINDOWS NO-PYTHON INSTALLER

Use this because your screenshot shows:
'Python was not found'

Steps:
1. Unzip this package.
2. Open the installer folder.
3. Double-click:
   run_clean_installer_windows_NO_PYTHON.bat
4. When it asks for the repo folder path, paste ONLY the folder path, not the python command.

Example of what to paste:
C:\Users\Matt Llewellyn\OneDrive\Documents\GitHub\f1-data-publisher

Do NOT paste:
python installer\install_f1_github_automation_clean.py --repo ...

The PowerShell installer will do the cleaning and install without Python.
