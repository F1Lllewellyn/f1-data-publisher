# F1 Actual Forecast Producer v1

Verdict: Pass with warnings.

This patch installs the missing upstream forecast-producing workflow for the forecast bundle chain.

It writes actual forecast rows into:

```text
latest/forecasts/<event_id>/<gate>/<lane>/forecast_rows.csv
```

and mirrors them into:

```text
latest/forecast_outputs/<event_id>/<gate>/<lane>/forecast_rows.csv
```

It supports all five gates and all three engine lanes:

- pre_weekend
- post_fp3
- post_qualifying
- race_result
- post_event

- stable_baseline
- control_room_overlay
- experimental_challenger

## Important guardrail

This workflow does not fabricate rows if no usable source rows exist. It requires at minimum usable driver or starting-grid source rows from the current source-closure outputs.

If no usable sources exist, it writes a runtime audit artifact and exits without committing forecast rows.

## Validation class

Rows generated by this workflow are actual generated forecast rows. They become blind-validation eligible only after Forecast Bundle Locker locks them before the relevant outcome is known.

F1 Actual Forecast Producer v1
