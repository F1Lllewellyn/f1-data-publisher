F1 Dashboard Connector One-Click Robust Repair

Purpose:
- Fixes the dashboard connector install failures caused by unquoted Windows paths and missing scripts.
- Uses PowerShell with quoted literal paths, so repo paths with spaces work.
- Installs the full payload: workflows, scripts, config, and schema.

Install:
1. Extract this ZIP to C:\F1Patch or another short folder.
2. Open F1DashRepair.
3. Double-click install.bat.
4. Press Enter for the default repo path.
5. Wait for INSTALL CHECK PASSED.
6. Commit and push in GitHub Desktop.
7. Run: F1 Forecast/Fantasy Readiness Dashboard - Safe Test Button.
