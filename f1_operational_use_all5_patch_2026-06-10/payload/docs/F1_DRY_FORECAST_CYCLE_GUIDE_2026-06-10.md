# Dry Forecast Cycle Guide

The dry forecast cycle tests whether Elite outputs can be consumed safely.

It generates:

- forecast_consumption_decision_board.csv
- qualifying_confidence_adjustments.csv
- race_reliability_risk_flags.csv
- fantasy_risk_value_notes.csv
- promotion_gate_review.csv
- forecast_dry_run_summary.json
- dry_forecast_cycle_report.md

No automatic race P1-P20, qualifying P1-P5, or fantasy transfer changes are allowed.
